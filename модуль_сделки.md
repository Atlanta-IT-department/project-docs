# Модуль Сделки
## Описание
В CRM Vector добавлен новый модуль Сделки, который позволяет пользователям с ролью директора или управляющего создавать сделки. Модуль поддерживает несколько типов сделок, которые можно использовать в процессе работы с недвижимостью.

## Логика работы
Пользователь с ролью директора или управляющего может получить доступ к модулю Сделки и создавать новые сделки. При создании сделки необходимо выбрать один из доступных типов сделок. 

## Поля / Параметры

| Поле                | Описание                                               |
|---------------------|--------------------------------------------------------|
| Тип сделки          | Тип сделки, который будет создан (аванс, расчет, частичное растражение, растражение) |

## Связи и Контекст
Модуль Сделки интегрирован с другими модулями CRM Vector, что позволяет корректно учитывать сделки в рамках общих процессов работы с недвижимостью. Пользователи с необходимыми правами имеют возможность управлять сделками и отслеживать их статус.
